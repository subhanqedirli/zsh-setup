# zsh-setup
Ultimate ZSH setup with oh-my-zsh and language packages
## Installing
```
git clone https://github.com/subhanqedirli/zsh-setup.git
cd zsh-setup-main
./install.sh
```
## Features
 - Install ZSH
 - Install oh-my-zsh
 - set ZSH as default Shell
 - Install oh-my-zsh plugins & themes
 - Installa languagepacks
